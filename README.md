# Mitochondria Industrial Energy Platform

Open-source engineering software for advanced industrial energy systems.

This project contains engineering tools and software used for the design and optimization of industrial thermodynamic systems.

## Main Areas

* Supercritical CO₂ Brayton cycle calculations
* Organic Rankine Cycle (ORC) modeling
* Absorption-compression hybrid systems
* Industrial transformer design tools
* Electric machine design utilities
* Process simulation calculators

## Purpose

The project aims to develop engineering software tools supporting the concept of the **Mitochondria Industrial Energy Platform**, a modular energy system for factories and industrial facilities.

The software tools included in this repository are intended for:

* industrial thermodynamic analysis
* energy efficiency optimization
* power electronics and electrical engineering calculations
* experimental energy system modeling

## Technologies

* Python
* JavaScript
* HTML engineering calculators
* Embedded control tools
